# opengl-sandbox

Learning OpenGL — experiments and exercises following learnopengl.com.

## Setup

Dependencies: GLFW, GLEW (via Homebrew)

\```bash
brew install glfw glew
mkdir build && cd build
cmake ..
make
\```
